Sistema web de gestión de pedidos de Gas de Oaxaca

Gas de Oaxaca es una plataforma web desarrollada para gestionar pedidos de gas LP a domicilio. El sistema permite a los clientes consultar productos, agregarlos al carrito, confirmar pedidos, seleccionar un método de pago y registrar su ubicación mediante un mapa para facilitar el proceso de entrega.

La aplicación cuenta con módulos para clientes, administradores y repartidores. El administrador puede gestionar productos, clientes, repartidores, pedidos, inventario y consultar información del negocio mediante indicadores. Por su parte, el repartidor puede visualizar pedidos asignados, dar seguimiento a las entregas y marcar pedidos como entregados.

Funcionalidades principales

Gestión de pedidos

Registro, confirmación y seguimiento de pedidos con estados como pendiente de pago, pagado y entregado.

Roles de usuario

Separación de funciones para clientes, administradores y repartidores, permitiendo controlar el acceso a cada módulo del sistema.

Dashboard administrativo

Visualización de ventas, pedidos pendientes, productos con bajo stock, pedido más alto y reportes básicos para la toma de decisiones.

Mapa y ubicación

Integración con Leaflet para capturar la ubicación del cliente y apoyar la logística de entrega de pedidos.
