Sistema de punto de venta: GADY Accesorios

GADY Accesorios es un sistema de punto de venta desarrollado para facilitar la administración de una tienda dedicada a la venta de productos y peluches. El sistema permite organizar las operaciones principales del negocio, como el registro de ventas, el control de inventario, la gestión de clientes, el manejo de vendedores y el registro de pagos.

El proyecto está diseñado para que el usuario pueda trabajar desde un menú principal, accediendo a distintos módulos según las tareas que necesite realizar. Su objetivo es mejorar el flujo de trabajo diario, reducir errores en los registros y mantener organizada la información comercial de la tienda.

Funcionalidades principales

Registro de ventas

Permite registrar las ventas realizadas, mostrando datos como ID de venta, fecha, vendedor, cliente, productos, cantidades, precios y total del pedido.

Control de inventario

Facilita la consulta de productos disponibles, búsqueda por código de barras, actualización de stock y control de existencias dentro del sistema.

Gestión de clientes y vendedores

Incluye módulos para registrar, consultar, modificar y eliminar información de clientes, así como administrar los datos de los vendedores del negocio.

Pagos y ticket de compra

Permite ingresar el monto pagado por el cliente, calcular el cambio, confirmar el pago y generar un ticket de compra con los detalles de la transacción.

Módulos del sistema

Inicio de sesión

Acceso mediante usuario y contraseña para proteger el sistema y permitir el ingreso al menú principal.

Menú principal

Panel central desde donde se accede a ventas, inventario, clientes, empleados, pagos y demás funciones administrativas.

Búsqueda y filtros

Permite localizar ventas, clientes o productos por distintos criterios, como ID, fecha, vendedor o código de barras.
